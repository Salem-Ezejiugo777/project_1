# Kosisochukwu Ezejiugo - Portfolio Website

A professional portfolio website built with HTML5, CSS3, and vanilla JavaScript. Features a responsive design, academic planner, and contact management system.

## 🚀 Features

- **Responsive Design** - Mobile-first approach works on all devices
- **Dynamic Navigation** - Hamburger menu for mobile devices
- **Academic Planner** - Add, complete, and manage academic tasks with local storage persistence
- **Contact Form** - Fully validated contact form with error handling
- **Modern Styling** - Clean, professional design with smooth animations
- **Accessibility** - ARIA labels, semantic HTML, keyboard navigation

## 🔧 How to Use

### Local Development
1. Clone the repository
2. Open `index.html` in your web browser
3. Navigate through pages using the menu

### Academic Planner
1. Enter task description in the input field
2. Optionally select a due date
3. Click "Add Task" to save
4. Tasks are automatically saved to browser storage
5. Mark complete or delete using buttons on task items

### Contact Form
1. Fill in all required fields (name, email, phone, message)
2. Form validates email format and phone numbers
3. Submit to see success confirmation
4. Email validation uses basic regex pattern
